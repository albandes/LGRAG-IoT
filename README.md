# IoT LGPD Compliance Dataset

Synthetic dataset designed to evaluate AI-based approaches for verifying compliance with the Brazilian General Data Protection Law (LGPD) in Internet of Things (IoT) environments.

## Overview

This dataset simulates data flows generated by IoT devices and their associated privacy metadata. Each record represents a data processing scenario that may or may not comply with LGPD principles.

The dataset was created to support research in:

- AI for privacy compliance
- IoT data governance
- Retrieval-Augmented Generation (RAG)
- Explainable AI for legal reasoning
- Automated compliance verification

## Dataset Characteristics

The dataset contains **500 synthetic IoT data processing scenarios**, each describing:

- device type
- type of personal data
- data subject
- processing purpose
- legal basis
- data sharing behavior
- storage location
- retention policy
- anonymization status
- consent status

Each scenario is labeled with the expected compliance status according to simplified LGPD rules.
